    부록 D: 유용한 리소스 링크 모음¶

    강의, 책, 커뮤니티, 온라인 저지 — 더 깊이 파고들고 싶을 때 찾아보세요.

    D-1. 한국 개발자 커뮤니티¶

    블로그 플랫폼¶

    • Velog (velog.io) — 개발자 전용, 기술 글 품질 높음
    • Tistory (tistory.com) — 개인 기술 블로그
    • Medium (medium.com) — 국제 기술 글, "Better Programming" 구독 추천
    • dev.to (dev.to) — 영문이지만 예제 풍부

    커뮤니티 & 채용¶

    • 프로그래머스 (programmers.co.kr) — 코딩테스트 + 채용
    • 백준 온라인 저지 (acmicpc.net) — 한국 알고리즘 허브
    • 인프런 (inflearn.com) — 한국어 온라인 강의, 가성비 최고
    • 노마드 코더 (nomadcoders.co) — 웹 개발 무료/유료 강의
    • OKKY (okky.kr) — 개발자 Q&A

    D-2. 온라인 저지¶

    사이트 언어 특징 추천 대상
    LeetCode 영문 상세 해설 외국계 지망
    Baekjoon 한국어 기출 많음 공기업/대기업
    Programmers 한국어 카카오/네이버 기출 취준생 필수
    Codeforces 영문 경쟁 프로그래밍 상급자
    AtCoder 일본어/영문 고난도 상급자

    💡 학습 전략 (3-4개월, 200문제)

    • Week 1-4: LeetCode Easy 50문제
    • Week 5-8: LeetCode Medium 50문제
    • Week 9-12: Baekjoon Silver 50문제
    • Week 13-16: Baekjoon Gold 50문제

    D-3. 무료 온라인 강좌¶

    범용¶

    • freeCodeCamp (freecodecamp.org) — 완전 무료, 영상 품질 최상
    • Responsive Web Design (5시간)
    • JavaScript Algorithms (4시간)
    • React (12시간)
    • Backend Development (12시간)
    • Khan Academy (khanacademy.org) — CS 기초
    • Codecademy (codecademy.com) — 인터랙티브
    • Udemy (udemy.com) — 세일 시 $10

    한국어¶

    • 인프런 (inflearn.com) — 한국 강사
    • 스파르타코딩클럽 (spartacodingclub.kr) — 입문자 친화
    • 생활코딩 (opentutorials.org) — 무료

    전문 기술¶

    • Docker 101 (docker.com/101-tutorial)
    • Kubernetes 공식 (kubernetes.io/docs/tutorials)
    • Spring 가이드 (spring.io/guides)
    • React 공식 (react.dev) — 2023 리뉴얼 버전

    D-4. 도서 추천¶

    필독서¶

    1. Cracking the Coding Interview — Gayle Laakmann McDowell (취업 준비 필수)
    2. System Design Interview — Alex Xu (시니어 준비)
    3. Clean Code — Robert C. Martin (코드 품질)
    4. Effective Java — Joshua Bloch (Java 심화)
    5. 실용주의 프로그래머 — 앤드류 헌트 (개발자 철학)

    알고리즘/자료구조¶

    • Introduction to Algorithms (CLRS) — 알고리즘 바이블
    • 프로그래밍 대회에서 배우는 알고리즘 문제해결 전략 (종만북) — 한국 저자
    • 코딩 인터뷰 완전분석 (Cracking 한국어판)

    인프라/DevOps¶

    • DDIA (Designing Data-Intensive Applications) — Martin Kleppmann
    • The Phoenix Project — DevOps 소설

    금융/도메인¶

    • 도메인 주도 설계 (DDD) — Eric Evans
    • 마이크로서비스 패턴 — Chris Richardson

    D-5. AI 개발 도구¶

    • Claude Code 공식 문서 (docs.claude.com/claude-code)
    • MCP 공식 (modelcontextprotocol.io)
    • GSTACK GitHub (github.com/garrytan/gstack)
    • Anthropic 공식 블로그 (anthropic.com/news)

    D-6. 뉴스레터 & 팟캐스트¶

    • GeekNews (news.hada.io) — 한국 IT 뉴스 큐레이션
    • Hacker News (news.ycombinator.com) — 글로벌 개발자 허브
    • TLDR (tldr.tech) — 일일 테크 요약 뉴스레터
    • 요즘IT (yozm.wishket.com) — 한국어 테크 매거진

    D-7. YouTube 채널¶

    한국어¶

    • 생활코딩 — 기초 입문
    • 노마드 코더 — 풀스택 + 최신 트렌드
    • 드림코딩 by 엘리 — 자바스크립트/CS
    • 조코딩 — 실습 위주

    영문¶

    • Fireship — 빠른 요약 스타일
    • Theo - t3.gg — 풀스택 최신 트렌드
    • ThePrimeagen — Vim / Rust / 실전

    체크리스트¶

    • 블로그 플랫폼 1개 정했다
    • 온라인 저지 1개에 가입했다
    • YouTube 채널 3개 구독했다
    • 도서 1권 구매/대여했다
    • Claude Code 공식 문서를 북마크했다
