## 부록 읽는 법

!!! tip "💡 꿀팁: 순서는 자유"
    본문과 달리 부록은 필요할 때만 점프해서 읽으면 됩니다. 북마크로 저장해두고 막힐 때마다 찾아오세요.

!!! info "📌 각 부록의 공통 구조"
    - "이걸 언제 써야 하는가"로 시작
    - 핵심 개념 + 다이어그램
    - 실전 예제 + 복사 가능한 명령어
    - 함정과 해결
    - 체크리스트
