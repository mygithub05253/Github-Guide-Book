    🌐 로드맵 포털¶

    roadmap.sh 커리큘럼을 기준으로 분야별 핵심 오픈소스 레포지토리를 카드형 UI로 탐색하고, 딥다이브 표준 템플릿을 통해 실전 활용법을 학습합니다.

    학습 방법

    각 카드는 "왜 쓰는가 → 10분 퀵스타트 → 코드 해부학 → AI 프롬프트 팩" 순서로 구성되어 있습니다. README만 읽는 것이 아니라, 바로 실행하고 AI에게 질문하는 실전 워크플로우에 초점을 맞춥니다.

    🔄 콘텐츠 수집 파이프라인 (MCP + GSTACK)¶

    각 딥다이브 문서는 다음 워크플로로 생성·유지됩니다.

    1. Fetch MCP로 공식 README.md 및 핵심 소스 수집
    2. Claude /think로 초안 작성 (표준 템플릿 적용)
    3. GSTACK /review로 품질 점검·보강
    4. docs/roadmap/**/index.md에 배치 후 mkdocs build --strict로 검증

    명령어 예시는 부록 · Plan 모드 가이드와 GSTACK 실전 통합 사례를 참조하세요.
